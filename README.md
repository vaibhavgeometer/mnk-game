# MNK Game Deluxe

A generalized version of Tic Tac Toe (m,n,k-game) built with Python and Pygame.

## Features
- **Customizable Board**: Choose board size (rows x cols) and win condition (k-in-a-row).
- **8 AI Levels**: Challenge AIs ranging from Elo 0 (Random) to Elo 2500 (Unbeatable Deep Search).
- **Time Controls**: Competitive play with adjustable thinking timers for both Human and AI.
- **Modern UI**: Clean interface with sliders and sound effects.

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the game:
   ```bash
   python main.py
   ```

## Controls
- **Mouse**: Click to place pieces / interact with UI.
- **Settings**: Adjust AI Level, Time Limit, and Board Rules in the settings menu.
